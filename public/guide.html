🗂️ 공공데이터포털 API 인증 가이드

소상공인 상가(상권)정보 API를 발급받는 과정을 단계별로 안내합니다.

1

공공데이터포털 접속 및 회원가입

https://www.data.go.kr 에 접속합니다.

  1. 우측 상단 회원가입 클릭
  2. 본인인증 진행 (휴대폰 또는 아이핀)
  3. 회원 유형은 일반회원 선택
  4. 아이디, 비밀번호, 이메일 입력 후 가입 완료
💡 팁: 이미 계정이 있다면 바로 로그인하세요.
2

로그인 후 API 검색

  1. 로그인 후 상단 검색창에 아래 키워드를 입력합니다:
소상공인시장진흥공단_상가(상권)정보
  1. 검색 결과에서 소상공인시장진흥공단_상가(상권)정보를 클릭합니다.
  2. 제공 기관이 소상공인시장진흥공단인지 확인합니다.
⚠️ 주의: 비슷한 이름의 데이터가 여러 개 있을 수 있습니다. 반드시 "오픈API" 탭이 있는 항목을 선택하세요.
3

오픈API 활용 신청

  1. 데이터 상세 페이지에서 오픈API 탭을 클릭합니다.
  2. 활용신청 버튼을 클릭합니다.
  3. 신청 양식을 작성합니다:
항목 입력 내용
활용 목적 "학습 및 교육 목적으로 카페 위치 정보를 지도에 표시"
서비스 유형 웹 서비스 개발
서비스 URL 배포 URL 입력 (예: https://my-app.onrender.com)
동의 항목 모두 체크
  1. 신청 버튼을 클릭합니다.
💡 팁: 승인은 보통 자동 승인되어 1~2시간 내에 완료됩니다.
4

API 인증키 확인

승인이 완료되면 인증키를 확인할 수 있습니다.

  1. 상단 메뉴 마이페이지 클릭
  2. 데이터 활용 → 활용신청 현황 클릭
  3. 신청한 API 항목을 찾아 클릭
  4. 일반 인증키 (Encoding)를 복사합니다

일반 인증키 (Encoding) 예시:

a1B2c3D4e5F6g7H8i9J0k1L2m3N4o5P6q7R8s9T0...
⚠️ 주의: 인증키는 Encoding된 키와 Decoding된 키 두 가지가 있습니다. Encoding 키를 사용하세요.
5

API 호출 테스트

인증키가 정상 작동하는지 브라우저에서 바로 테스트할 수 있습니다.

아래 URL에서 YOUR_API_KEY 부분을 본인의 인증키로 교체한 후, 브라우저 주소창에 붙여넣기 하세요:

https://apis.data.go.kr/B553077/api/open/sdsc2/storeListInDong?serviceKey=YOUR_API_KEY&divId=indsSclsCd&key=I212&pageNo=1&numOfRows=5&type=json

위 URL은 업종코드 I212 (커피점/카페)로 카페 데이터를 5개 조회하는 예시입니다.

💡 성공 시: JSON 형태의 카페 데이터(상호명, 주소, 좌표 등)가 브라우저에 표시됩니다.
⚠️ 에러 시: "SERVICE_KEY_IS_NOT_REGISTERED_ERROR"가 뜨면 아직 승인 대기 중입니다. 1~2시간 후 다시 시도하세요.
